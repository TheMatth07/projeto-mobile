# Catálogo Mobile

Projeto: Catálogo Mobile

Aplicativo desenvolvido em React Native utilizando Expo.

Funcionalidades:
- Login de usuário com validação
- Listagem de produtos consumindo API
- Visualização de detalhes do produto
- Carrinho de compras
- Filtro de produtos por categoria
- Cálculo do total do carrinho
- Logout

## Tecnologias
- React Native
- Expo
- Axios
- React Navigation


## API utilizada
https://dummyjson.com/products

## Como executar

git clone
npm install
npx expo start
